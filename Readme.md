# QuickDice


这是一个简便的掷骰子程序  
输入骰子个数与值的范围（最大值），然后开始掷骰子  
也可以通过表达式识别来指定骰子个数与范围  

如果表达式识别栏里有东西，优先按表达式来识别  
如果那一栏是空的，再按上面单独输入的来算  



## 表达式格式：
* `xdy`：掷x次最大值为y的骰子
* `dx`：掷1次最大值为x的骰子
* `x（数字）`：固定数字x

**以上三种表达式可通过加号+进行连接**

### 表达式示例：
* `1d5`	掷1次最大值为5的骰子

* `4d6`	掷4次最大值为6的骰子

* `d7`	掷1次最大值为7的骰子

* `64`	固定数字64

* `8d3+d6+5`	掷8次最大值为3的骰子+掷1次最大值为6的骰子+固定数字5

* `d9+7+2d5+d6`	掷1次最大值为9的骰子+固定数字7+掷2次最大值为5的骰子+掷1次最大值为6的骰子
